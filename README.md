# Telegram Chat Analytics (PersonaAI)

Инструмент для автоматизированного сбора, очистки и семантического анализа неструктурированных текстовых данных из дампов Telegram. Проект решает задачу извлечения количественных метрик и инсайтов из сырых выгрузок переписок.

## Про архитектуру

Стандартные выгрузки истории чатов (`result.json`) часто бывают повреждены при экспорте больших объемов данных, что делает невозможным их обработку стандартными JSON-парсерами. 

Проект вытягивает данные даже из поврежденных выгрузок Telegram, очищает их от системного мусора и превращает в готовый для работы набор сообщений и авторов.

### Основной функционал:
1. **Очистка и подготовка данных:** Превращает грязный и битый JSON в чистый датасет, готовый к анализу.
2. **Базовые метрики активности:** Считает частоту сообщений и объем текста по каждому участнику.
3. **Семантический анализ текста (DeepSeek API):**
   * Оценка общей тональности и атмосферы в чате.
   * Составление кратких профилей участников на основе их текстов.
   * Нахождение конфликтов и предложения по их решению.

## Требования к окружению
Для запуска требуются валидные ключи доступа:
1. `BOT_TOKEN`
2. `DEEPSEEK_API_KEY`
